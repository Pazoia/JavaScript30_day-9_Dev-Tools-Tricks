In this lesson I covered some commands to be applied to the console.
Commands such as:
  • console.log('hello');

    this is the mostly used command, i'll print whatever is inside the parentheses.

  • console.log('Hello, I am a %s string', 'ðŸ’©');

    using interpolation the %s will be replaced by whatever is inside the quote marks after the comma, this can be used to add a variable for example.

  • console.log('%c I am some cool text!', 'font-size:25px; color:red;');

    adding %c at the beggining of the text inside the parentheses, followed by styling requirements after the comma, the text printed in the console will be formatted to the styling instructions.

  • console.warn('OH NOOOO');

    .warn(); will return a warning message on the console, the message will be the text inside the parentheses.

  • console.error('Broken!!!');

    .error(); will return an error message on the console, the message will be the text inside the parentheses.

  • console.info('Crocodiles eat 3-4 people per year');

    .info(); will return a message on the console with the 'i' symbol next to it, the message will be the text inside the parentheses.

  • console.assert(1 === 2, 'This is wrong!');

    .assert(); will return an error message if the assertion is false, the message being displayed is written after the comma.

  • console.clear();

    .clear(); will clear the console.

  • console .log(p);

    you can use .log(p); to print the element inside the parentheses in the console.

  • console.dir(p);

    .dir(p); will display on the console all the properties of the element inside the parentheses.

  • console.group(`${dog.name}`);
    console.log(`This is ${dog.name}`);
    console.log(`${dog.name} is ${dog.age} years old`);
    console.log(`${dog.name} is ${dog.age * 7} dog years`);
    console.groupEnd(`${dog.name}`);

    the text betwen .group(); and .groupEnd(); will all be inside a harrow header on the console, in this case the header will be the dog's name.
    To make it neater and with less info opened use .groupCollapsed(); instead of .group(); this will collapse the info in the arrows.

  • console.count('Paulo');
    console.count('Helen');
    console.count('Paulo');
    console.count('Helen');

    .count(); will count real life how many times something has been repeated, in this case it'll print in the console the following:
    Paulo: 1
    Helen: 1
    Paulo: 2
    Helen: 2

  • console.time('fetching data');
    fetch('https://api.github.com/users/wesbos')
    .then(data => data.json())
    .then(data => {
    console.timeEnd('fetching data');
    console.log(data);
    });

    using .time(); will print the time taken to perform a task, in the example above it'll print the time it took to fetch wes bos user data from github.

  • console.table(dogs);

    .table(); will organize the data from an array into a table in the console.
